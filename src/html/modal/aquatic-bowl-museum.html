Vue.JS project used to create animal profiles which can be navigated through based on the Discover Zone allocated in data JSON file. Created accordion, mouse hover functionalities on images, Meta-data structure for Analytics purposes in future and many more using Vue.JS.

technologies used:

Vue.JS, Vue Carousel, Vue Fullpage.js, Vue Match Heights, Vue Meta, Vue Scroll To, Vue Router, Bulma, SASS and Responsive Web Design
